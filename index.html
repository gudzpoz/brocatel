Getting Started
Road Map

Easy to Use

It is just Markdown (with Lua variables). Familiar with Markdown headings, links and lists? You're good to go!

Portable

Lua being Lua, it runs on most devices that you can possibly imagine, even in browsers and consoles.

Localization

Designed with internalization needs in mind. A context-rich POT file is generated upon compilation for integration with GNU gettext.

Fully Customizable

The macro system allows literally any compile-time manipulation of the Markdown structure.
